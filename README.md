## yuu-js

**yuu-js** は、Discord サーバー運営をサポートする日本語向け多機能ボットです。  
レベル・経済・チケット・ロールパネル・管理系ユーティリティなど、コミュニティ運営に必要な機能をひとまとめに提供します。

### 主な機能

- **レベリング / ランキング**
    - メッセージに応じた XP 付与
    - `/rank` で自分のレベル確認
    - `/leaderboard` でランキング表示

- **サーバー情報**
    - `/server` でサーバー概要を表示
    - `/user` でユーザー情報を表示

- **チケットシステム**
    - `/ticketpanel` でサポート用パネルを作成
    - パネルのボタンから専用テキストチャンネルを自動生成

- **ロールパネル**
    - `/rolepanel` でロール付与用パネルを作成・編集・再投稿
    - ボタン形式 / セレクトメニュー形式を選択可能

- **管理・モデレーション**
    - `/kick`, `/ban`, `/timeout`, `/purge` などの基本的な管理コマンド
    - `/permission` でチャンネル/ロール権限を GUI 的に編集

- **設定**
    - `/config` グループでボットの挙動や役職などをサーバーごとにカスタマイズ

### 動作環境

- Node.js / Bun が動作する環境
- Discord Bot アカウント（トークン）
- LibSQL / SQLite データベースファイル（このリポジトリでは `data/bot.db` を使用）

### セットアップ

1. 依存関係のインストール

    ```bash
    bun install
    ```

2. `.env` の作成（`DISCORD_TOKEN` などを設定）

3. スキーマ初期化は起動時に自動で行われます。  
   既存 DB を流用する場合は、必要に応じて `lib/schema.ts` の定義に合わせてマイグレーションしてください。

4. ボットの起動

    ```bash
    bun run index.ts
    ```

### 開発メモ

- スラッシュコマンドは `commands/` 以下にカテゴリごとに配置されています。
- 永続インタラクション（ボタン・セレクト）のハンドラは `lib/handlers/` 以下にあります。
- 新しい機能を追加する際は、既存のコマンド実装 (`commands/admin/*`, `commands/utility/*` など) を参考にしてください。
